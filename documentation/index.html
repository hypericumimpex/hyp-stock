Stock Synchronization for WooCommerce

Get your inventory in sync!

  • created: 02/01/2017
  • latest update: 3/12/2018
  • by: Max Sebes

Stock Synchronization for WooCommerce is build to make your eCommerce life easier! If your warehouse or supplier provides stock information in CSV format, you can now get your stock and website in sync!

Installation

1. Place the pluginfiles (folder called stock-synchronization) in the WordPress plugin folder (/wp-content/plugins/) or upload a .ZIP file containing the folder to your WordPress installation.

2. Activate the plugin. 

3. Navigate to WooCommerce > Settings > Products > Stock Synchronization

4. Fill in the information in the Stock Synchronization tab and save your data.

5. Your stock information will now be in sync 

Usage

Stock Synchronization for WooCommerce will automatically synchronize your stock information, no further steps are needed when in use! 

FAQ

• What if my stock information doesn't sync after filling in the settings?

Maybe your stock information doesn't get synced directly after installing the plugin, try waiting the time for the selected update frequency or manually sync by using this URL:

http://yourdomain.com/wp-cron.php?stocksync_cron_hook

Another way to manually run the synchronization is through the free plugin WP Crontrol. When installed, go to Tools > Cron Events and search for the stocksync_cron_hook and run that function. Your stock should now be syncing. 

• What if I don't want to use wp-cron? 

That's OK, just disable WP_CRON in your wp-config.php file and add the following URL to your own cronjobs: 

http://yourdomain.com/wp-cron.php?stocksync_cron_hook

• Can I also use XML-files?

No, XML-files are not supported (yet). 

• I save my CSV file locally, what now? 

Just fill in the link to your file (e.g. http://yourdomain.com/feed.csv).

• What if my CSV file doesn't have a top row with column headers? 

That is no problem, just fill in # with your coloumn number. For example, if your SKU is in the third column, fill in #3 in the SKU-field.

• How do I use the plugin for Variation products?? 

When your Product Variations have the same SKU as your main product, you can check the box 'Use for Variations'. Then you can fill in the Product ID or Article number from your CSV file in the table at the bottom. Then set the Variation ID with the corresponding Product ID and your stock will be in sync!

• My file is located on Dropbox

Dropbox files are supported, just make sure your Dropbox file is public and copy the Dropbox link. Submit this link in the "Link to CSV file" field. Your link should look like this:

https://www.dropbox.com/s/ID/file.csv?dl=1

Notice the ?dl=1 at the end of the URL, the default is ?dl=0, but you must change it to ?dl=1 in order to make it work!

• My file is located on Google Drive

Google Drive files are supported, just make sure the file is public and copy the Google Drive link. The Google Drive URL should look like this: 

https://docs.google.com/document/d/FILE_ID/edit?usp=sharing

Copy the FILE_ID and insert it into the "Google Drive File ID" field, check the "Google Drive File" box and get synced!

• Does the plugin have a log?

Log files are available under Tools / Stock Synchronization Log

• Is there a sample .csv file?

Yes, the sample .csv file can be found in the /documentation folder.
